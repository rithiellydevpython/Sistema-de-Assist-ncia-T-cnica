Sobre o projeto

Este projeto esta em desenvolvimento com o objetivo de criar um sistema de gerenciamento de serviços para uma loja de conserto de celulares. A aplicação permitirá o cadastro de serviços, gerenciamento de clientes e controle financeiro entre outras funcionalidades. estou começando a desenvolver o front-end no dia 18/01/2026 sem previsao de termino. A princio vai ser algo bem simples somente com tela de login, secção sobre, home e contato. depois do front-end pronto vou iniciar a responsividade e depois o back-end. E futuramente todas as funcionalidades que uma loja de conserto de celulares precisa. obs: estou fazendo esse projeto sozinho para aprender mais sobre desenvolvimento web. entao peço a compreensão sobre erros de iniciante principalmente na parte do front ja que estou focando minha carreira no back-end.
